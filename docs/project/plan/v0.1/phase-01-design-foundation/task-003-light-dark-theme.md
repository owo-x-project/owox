# Task 003: ライト/ダーク CSS 変数セット + テーマ切替ロジック

## 目的

ライトテーマとダークテーマの CSS 変数セットを作成し、OS 追従 + 手動切替の仕組みを実装する。

## 前提条件

- Task 002 (デザイントークン) が完了している

## 作業内容

- ライトテーマ用の CSS 変数セットを定義
  - 背景: 白〜淡グレー系
  - テキスト: ダークグレー〜黒
  - ボーダー: 淡いグレー
  - アクセント: owox ブルー
- ダークテーマ用の CSS 変数セットを定義 (既存の Caelestia 風を owox ブランドカラーへ調整)
- `prefers-color-scheme` メディアクエリでデフォルト自動検出
- テーマ状態を SolidJS ストアで管理 (`auto` / `light` / `dark`)
- `<html>` に `data-theme="light|dark"` を設定
- `localStorage` にユーザー選択を永続化
- テーマ切替トグルコンポーネントを作成 (ヘッダーバーに配置)
- FOUC 対策: `<head>` 内でテーマ初期設定スクリプトを実行

## 完了条件

- OS テーマ変更に自動追従する
- 手動トグルでライト⇔ダーク切替が即座に反映される
- リロード後もユーザー選択が維持される
- テーマ切替時にちらつきがない

## 検証方法

- Chrome DevTools で `prefers-color-scheme` をエミュレートして追従確認
- トグルボタンでの切替操作確認
- リロード後の設定保持確認

- テーマ切替ストア (`src/features/shell/theme.ts` 等)
