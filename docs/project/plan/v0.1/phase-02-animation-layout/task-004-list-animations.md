# Task 004: リスト項目追加/削除のアニメーション

## 目的

ファイルツリー、セッション一覧、Git ステータス一覧等のリストで項目の追加/削除時にスライド/フェードアニメーションを適用する。

## 前提条件

- Phase 01 完了

## 作業内容

- SolidJS `<TransitionGroup>` を使用したリストアニメーション
- 追加: slide-down + fade-in (120ms)
- 削除: slide-up + fade-out (120ms)
- 対象リスト:
  - ターミナルセッションタブ
  - ファイルツリーのディレクトリ展開/折りたたみ
  - Git ステータスのファイル一覧 (stage/unstage 移動)
  - エラートースト
- `prefers-reduced-motion` で無効化

## 完了条件

- 対象リストで項目追加/削除時にアニメーションがある
- アニメーション中にレイアウト崩れがない

## 検証方法

- ファイルツリーのディレクトリ展開/折りたたみでアニメーション確認
- Git の stage/unstage でファイルの移動アニメーション確認

## 依存関係

- Phase 01 完了
