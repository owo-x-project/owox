# Task 001: グラスモーフィズム適用

## 目的

ヘッダー、ドロワー、モーダル、シートにすりガラス効果を適用し、owox ブルー系の半透明パネルでリッチな視覚表現を実現する。

## 前提条件

- Phase 01 完了。テーマトークンが使用可能

## 作業内容

- ヘッダーバーに `backdrop-filter: blur(16px)` + 半透明背景を適用
- 左右ドロワー (プロジェクト一覧、ソースコントロール) に同様の効果
- CommandLauncher モーダルにすりガラスオーバーレイ
- ConfirmDialog のバックドロップにブラー強化
- Sheet コンポーネントのバックドロップにブラー追加
- ライト/ダーク両テーマで適切な透過度を調整
- `@supports (backdrop-filter: blur())` でフォールバック定義

## 完了条件

- 主要パネルにすりガラス効果が適用されている
- `backdrop-filter` 非対応ブラウザでもレイアウトが崩れない
- ライト/ダーク両方で視認性が確保されている

## 検証方法

- Chrome/Firefox/Safari でのすりガラス表示確認
- `@supports` フォールバックの動作確認

## 依存関係

- Phase 01 完了

## 成果物

- `src/styles.css` (変更)
- 各 feature CSS ファイル (変更)
