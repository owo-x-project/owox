# Task 002: マイクロインタラクション強化

## 目的

ボタン、タブ、リンク等のインタラクティブ要素にスムーズなホバー/プレス/フォーカスエフェクトを追加する。

## 前提条件

- Phase 01 完了

## 作業内容

- ボタンホバー: `scale(1.02)` + shadow 拡大 + 背景色変化 (160ms ease-standard)
- ボタンプレス: `scale(0.98)` + shadow 縮小 (80ms)
- タブ切替: アクティブインジケーターのスライドアニメーション
- リンクホバー: 下線の左→右スライドイン
- フォーカスリング: owox ブルーのアウトラインにフェードイン
- アイコンボタン: ホバー時にサブトルな glow 効果
- 入力フィールド: フォーカス時にボーダーカラーアニメーション
- 全マイクロインタラクションを `--motion-micro` / `--motion-standard` 変数で統一
- `prefers-reduced-motion` で全無効化

## 完了条件

- 全インタラクティブ要素にホバー/プレスエフェクトがある
- アニメーションが `--motion-*` 変数を使用している
- reduced-motion で無効化される

## 検証方法

- 全画面のボタン・タブ・入力フィールドでインタラクション確認
- `prefers-reduced-motion: reduce` で無効化確認
