# Requirements v1

## 役割

v1 は v0 で対象外または将来拡張とした推奨機能を扱う要求正本です。v0 の核を前提に、AI CLI Session、Context Governance、Task Graph、Policy、Workbench、Runtime、Repo、基盤、Open Format を強化する。

- `REQ-ai-cli-session-expansion.md`: AI CLI Session と adapter の拡張
- `REQ-context-governance-expansion.md`: Context Governance の強化
- `REQ-task-graph-planner-lite.md`: Task Graph / Planner-lite
- `REQ-policy-configuration.md`: 設定可能な Policy
- `REQ-workbench-surface-expansion.md`: Workbench 画面と確認面の拡張
- `REQ-runtime-governance-agents.md`: Governance Agent による判断補助
- `REQ-repo-management-expansion.md`: Repo 管理モードの拡張
- `REQ-platform-scaleout.md`: 個人利用を超える基盤拡張
- `REQ-open-format-publication.md`: Open Format 公開
