# Task 002: エラーログパネル

## 目的

過去に発生したエラー/警告を一覧で確認できるパネルを実装する。

## 前提条件

- Task 001 (トーストシステム) が完了している

## 作業内容

- エラーログストアの作成:
  - 全トーストを履歴として保持 (最大 100 件)
  - タイムスタンプ付き
  - 未読カウント管理
- エラーログパネル UI:
  - ヘッダーバーにベルアイコン + 未読バッジ
  - クリックでパネル展開 (右ドロワーまたはモーダル)
  - 時系列で一覧表示
  - 種別フィルタ (error/warning/info/success)
  - 個別削除 + 全クリア
- パネル展開時に未読カウントをリセット
- ログ項目のスタイリング: 種別アイコン + 色 + タイムスタンプ + メッセージ

## 完了条件

- ヘッダーにベルアイコン + 未読バッジがある
- クリックでエラーログ一覧が表示される
- フィルタで種別ごとに絞り込める
- 個別削除・全クリアができる

## 検証方法

- エラー発生後にベルアイコンの未読バッジを確認
- パネル展開で履歴一覧を確認
- フィルタ・削除操作を確認
