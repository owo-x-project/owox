# Task 005: ショートカットヘルプパネル + ツールチップ表示

## 目的

ショートカットの発見性を向上させる。ヘルプパネルとツールチップを実装する。

## 前提条件

- Task 004 (OS 検出) が完了している

## 作業内容

- ショートカットヘルプパネル:
  - `?` キーまたはメニューから開く
  - 全ショートカットをカテゴリ別に一覧表示
  - カテゴリ: 一般、ターミナル、エディタ、Git
  - 各ショートカットにキー表記 + 説明
  - モーダル or シートとして表示
- ツールチップ:
  - ショートカットを持つボタンのホバー時に表示
  - 遅延表示 (500ms)
  - 位置: ボタンの下
  - ショートカット表記を含む
- 頻度低いショートカットの整理:
  - 本当に必要なものだけキーボードショートカットを維持
  - それ以外はメニュー/コマンドパレットからのアクセスに統一

## 完了条件

- `?` キーでショートカットヘルプが表示される
- 主要ボタンのツールチップにショートカットが含まれる
- 不要なショートカットが廃止されている

## 検証方法

- `?` キーでヘルプパネル表示を確認
- ボタンホバーでツールチップ表示を確認
