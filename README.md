# Owox

Owox is a self-hosted Context / Work / Evidence control plane for AI Agent and AI coding CLI workflows.

## Project Documents

- `.agents/project.md`: AI working constraints and project summary.
- `.agents/tasks/`: AI task context caches and work logs.
- `docs/project/index.md`: human-facing project canon entry point.

## Current State

The repository harness is initialized. Product implementation code is not scaffolded yet.

## Validation

Run harness validation after changing `.agents/` or `docs/project/`:

```bash
rtk bash .agents/scripts/validate_harness.sh
```
