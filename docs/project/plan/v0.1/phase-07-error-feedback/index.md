# Phase 07: エラー・フィードバック改善

## 目的

エラー/警告の表示を改善し、トースト自動消滅、手動 dismiss、スタック表示、色分け強化、エラーログパネルを実装する。

## 前提条件

- Phase 01 (デザイン基盤) が完了している
- Phase 02 (アニメーション) のリストアニメーションが利用可能

## 完了条件

- 非致命エラーがトーストとして表示され、数秒で自動消滅する
- 全エラーに × ボタンがある
- 複数エラーがスタック表示される
- error/warning/info/success が視覚的に明確に区別できる
- 過去のエラーを一覧確認できるパネルがある

## 検証方法

- エラー発生時にトーストが表示されることを確認
- 自動消滅のタイミング確認
- × ボタンで手動 dismiss 確認
- 複数エラー同時発生時のスタック表示確認
- エラーログパネルの表示確認
