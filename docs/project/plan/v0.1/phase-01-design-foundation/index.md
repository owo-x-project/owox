# Phase 01: デザイン基盤

## 目的

UnoCSS を導入し、owox ブランドカラーに基づくライト/ダークテーマトークンを定義。既存バニラ CSS からの段階的移行基盤を構築する。

## 前提条件

- 既存のバニラ CSS + CSS 変数構成が動作している
- `src/styles.css` の CSS 変数体系が把握済み

## 完了条件

- UnoCSS が Vite に統合され、ビルドが通る
- owox ブランドカラー (ブルー〜シアングラデーション) のデザイントークンが定義済み
- ライト/ダーク両テーマの CSS 変数セットが存在する
- OS の `prefers-color-scheme` に追従し、手動切替トグルも動作する
- テーマ切替時にちらつきがない (FOUC 対策)
- 既存コンポーネントがテーマ切替後も崩れない

## 検証方法

- `vite build` が成功する
- ブラウザで OS テーマ変更に追従することを確認
- 手動トグルでライト⇔ダーク切替を確認
- 全画面でレイアウト崩れがないことを目視確認
