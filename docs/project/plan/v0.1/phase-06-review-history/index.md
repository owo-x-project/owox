# Phase 06: Review・コミット履歴

## 目的

過去のコミット履歴を一覧表示し、選択したコミットの diff を閲覧できるようにする。

## 前提条件

- Phase 01 (デザイン基盤) が完了している
- バックエンドに git log API が存在する (なければ追加が必要)

## 完了条件

- コミット履歴が時系列で一覧表示される
- コミットを選択すると diff が表示される
- ページネーション (無限スクロールまたは Load More) がある

## 検証方法

- コミット一覧が表示されることを確認
- コミット選択で diff が表示されることを確認
- 大量コミットのあるリポジトリでページネーション確認
