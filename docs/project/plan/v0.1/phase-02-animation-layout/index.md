# Phase 02: アニメーション・レイアウト磨き込み

## 目的

グラスモーフィズム、マイクロインタラクション、ページ遷移、リストアニメーションを追加し、レイアウトのジッター・崩れを解消する。

## 前提条件

- Phase 01 (デザイン基盤) が完了している
- ライト/ダーク両テーマの CSS 変数が使用可能

## 完了条件

- パネルにすりガラス効果 (`backdrop-filter: blur()`) が適用されている
- ボタン・タブのホバー/プレス時にスムーズなマイクロインタラクションがある
- 画面切替 (Terminal/Files/Review) にトランジションがある
- リスト項目の追加/削除にスライド/フェードアニメーションがある
- ホバー時のジッターやレイアウト崩れがない
- `prefers-reduced-motion` 時に全アニメーションが無効化される

## 検証方法

- 各画面でホバー・クリック操作時のアニメーションを目視確認
- 画面遷移のスムーズさを確認
- `prefers-reduced-motion: reduce` でアニメーション無効化を確認
- モバイル・タブレットビューポートでの崩れ確認
