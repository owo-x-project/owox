# Task 002: 隠しファイル半透明表示 + 表示/非表示トグル

## 目的

ドットファイル (.gitignore, .env 等) を半透明で表示し、ワンタッチで非表示にできるトグルを追加する。

## 前提条件

- Phase 01 完了

## 作業内容

- `tree-model.ts` に隠しファイル判定ロジック追加:
  - ファイル名が `.` で始まるものを隠しファイルとして扱う
- `FileTree.tsx` に隠しファイル表示制御:
  - 隠しファイル行に `opacity: 0.5` を適用
  - トグルボタンをツリーヘッダーに配置 (目のアイコン)
  - トグル状態: 「半透明表示」↔「完全非表示」
  - デフォルト: 半透明表示
- トグル状態を `localStorage` に永続化
- `files.css` に `.file-tree__row--hidden` スタイル追加

## 完了条件

- 隠しファイルが半透明で表示される
- トグルボタンで非表示に切替できる
- リロード後もトグル状態が維持される

## 検証方法

- `.gitignore`, `.env` 等が半透明で表示されることを確認
- トグルで非表示になることを確認
- リロード後の状態保持確認
